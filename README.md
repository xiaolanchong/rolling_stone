
# Rolling Stone

Universiity of Alberta's solution to Sokoban game. See https://webdocs.cs.ualberta.ca/~games/Sokoban/

## Maze format

 \# - wall
 @ - man
 +, % - man on a goal
 $, A-Z - crate (stone)
 \*, a-z - crate (stone) on a goal
 . - goal
 =, <space> - empty place

Maximum goals: 34
Maximum crates: 34
Maximum maze width: 20
Maximum maze height: 16
